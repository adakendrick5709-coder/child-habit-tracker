# 小小习惯侠 - 儿童习惯追踪网站

## 1. Concept & Vision

**"小星星收集计划"** — 孩子每完成一项任务，就获得一颗星星，攒够一定数量可以兑换小奖励。界面像一个卡通冒险地图，每天是地图上的一个新关卡，完成任务就能点亮星星。目标是让孩子觉得"打卡"是玩游戏，而不是做作业。

整体感觉：**明亮、活泼、有成就感**，像是在玩一个每日任务游戏。

---

## 2. Design Language

### 🎨 美学方向
卡通冒险风格，明亮饱和的色彩，大圆角卡片，无棱角设计，大量使用 emoji 作为情绪和奖励的表达。

### 🌈 色彩系统
| 用途 | 颜色 | Hex |
|------|------|-----|
| 主色（冒险蓝）| 蓝 | `#4A90D9` |
| 强调（星星金）| 金黄 | `#FFD93D` |
| 成功（完成绿）| 绿 | `#6BCB77` |
| 情绪（开心粉）| 粉红 | `#FF6B9D` |
| 背景（云朵白）| 米白 | `#FFF9F0` |
| 卡片底 | 纯白 | `#FFFFFF` |
| 文字 | 深灰 | `#2D3436` |

### ✏️ 字体
- 标题：`"ZCOOL KuaiLe"`（站酷快乐体）— 童趣感
- 正文：`"Noto Sans SC"` — 清晰易读
- 数字/星星计数：`"Fredoka One"` — 圆润可爱

### 📐 布局节奏
- 平板竖屏（768px-1024px）为主设计目标
- 卡片大圆角（24px），间距宽松（16-24px）
- 按钮高度 ≥ 56px（手指友好）
- 大量留白，不拥挤

### ⚡ 动效哲学
- 完成任务：星星弹入 + 放大缩小 + 轻微震动
- 情绪选择：emoji 弹跳效果
- 页面加载：卡片依次淡入（stagger 100ms）
- 奖励解锁：星星雨 + 徽章翻转动画
- 所有动画时长 200-500ms，使用 ease-out 或弹簧曲线

---

## 3. Features

### ✅ 核心功能
1. **5个习惯维度**：阅读、运动、情绪、表达、作业
2. **星星收集**：每完成一项获得1-3颗星
3. **成就徽章**：30颗→小富翁，50颗→坚持之星，100颗→超级习惯侠
4. **情绪选择**：5个emoji，早晚各一次
5. **提醒通知**：浏览器Notification API + 应用内倒计时条
6. **周报统计**：7天完成率柱状图
7. **数据导出**：JSON备份

## 4. Technical

- 单文件 HTML（HTML + CSS + JS）
- LocalStorage 持久化
- Notification API 浏览器提醒
- Google Fonts：ZCOOL KuaiLe + Noto Sans SC + Fredoka One
